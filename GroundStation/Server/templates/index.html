Generación de gráficos desde copia de seguridad

Este módulo permite la recuperación y análisis de datos históricos almacenados en el sistema del CanSat. Al iniciar el proceso, se establece una conexión segura mediante SSH con el dispositivo, siempre que se encuentre accesible dentro de la misma red local.

Los archivos de registro son transferidos a un entorno de procesamiento local, donde se analizan y transforman en representaciones gráficas. Esto permite estudiar el comportamiento del sistema en intervalos específicos o a lo largo de toda la misión.
