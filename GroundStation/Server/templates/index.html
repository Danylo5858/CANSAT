RECUPERACIÓN DE REGISTROS

Extracción remota de datos del sistema embarcado mediante conexión segura y procesamiento local para la generación de visualizaciones analíticas. El sistema se conecta al CanSat dentro de la misma red y accede a los registros almacenados durante la misión. Estos datos incluyen información de telemetría y estado del sistema, recogida a lo largo del vuelo. Una vez transferidos, los datos se procesan localmente para organizarlos y convertirlos en gráficos claros y útiles. Esto permite entender mejor cómo se ha comportado el sistema en distintos momentos. Puedes seleccionar un intervalo concreto para centrar el análisis en una fase específica, o procesar todos los datos disponibles para obtener una visión completa de la misión.
